YAML:

YAML Ain't Markup Language

- Is a human-readable data serialsation standard that is often used for configuration files
    - Key-Value Pairs
        - key: value

    - List
        - Lists:
            - item1
            - item2
            - item3
    
    - Nested elements
        - parent:
            child: value

YAML Syntax:

```yaml
# key value pairs

name: Foo Bar
age: 30
job: Developer

# Lists

fruits:
- apple
- banana
- cherry

# nested elements

address:
    street: 123 Downing street
    city: London
    country: United Kingdom
```
